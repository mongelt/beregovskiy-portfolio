## 🎨 How to Use in Editor

### Creating Blocks
1. Click the **+** button on the left
2. Select from menu:
   - Text: Paragraph, Header, Quote, Warning, Alert
   - Lists: List (can be nested)
   - Media: Image, Embed, Attaches
   - Layout: Table, Delimiter, Button
   - Code: Code, Raw HTML

### Inline Formatting
1. Select text
2. Toolbar appears with:
   - **B** - Bold
   - **I** - Italic
   - **🔗** - Link
   - **<>** - Inline Code
   - **🖍️** - Marker/Highlight
   - **U** - Underline

### Features
- **Drag & Drop:** Grab the ⋮⋮ handle to reorder blocks
- **Undo/Redo:** Press Ctrl+Z / Ctrl+Shift+Z (or Cmd on Mac)
- **Tab:** Increase nesting in lists
- **Shift+Tab:** Decrease nesting in lists
